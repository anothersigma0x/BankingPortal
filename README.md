# BankingPortal

Aplicación simple de banca automatizada en JavaScript que simula depósitos, retiros y consultas de saldo con verificación de PIN. La salida se muestra en la consola.

## Cómo usar
1. Descarga o clona este repositorio.
2. Asegúrate de tener Node.js instalado.
3. Abre una terminal en la carpeta del proyecto.
4. Ejecuta `node script.js`.
5. La consola mostrará operaciones de prueba con nombre "Jane", PIN 1568, depósito de $50, retiro de $1200, y un retiro inválido de $6600.

## Archivos
- `script.js`: Lógica para operaciones bancarias en consola.

## Requisitos
- Node.js.
